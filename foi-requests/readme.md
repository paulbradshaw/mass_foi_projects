# FOI requests

This folder contains example FOI requests you can use to test the processes, tools and techniques outlined in the repo.

You can also use FOI responses published on WhatDoTheyKnow: [this search](https://www.whatdotheyknow.com/search/Council%20football%20pitch%20fees%20and%20number/all), for example, will return FOI requests from Birmingham City University journalism students for a project investigating council pitch fees.
